[B] 중복 감지 & 정리
  - dHash 기반 중복 사진 자동 탐지
  - 중복 그룹 리스트 → 사용자가 검토 후 일괄 삭제 (간편 삭제 버튼)
  - 삭제 시 기기 용량 확보량 표시

[C] 유사 사진 Best Cut 추천 (제미나이)
  - 비슷한 구도 사진 그룹에서 가장 선명/표정 좋은 1장 추천

[D] 타임라인 자동 정렬
  - EXIF 날짜 기반 시간순 정렬

    → 정리 전 혼란스러운 타임라인 vs 정리 후 깔끔한 타임라인 대조

[E] AI 인물별 앨범
  - face-api.js 브라우저 내 인물 인식 (서버에 사진 전송 X)
  - 인물 태그 → 자동 앨범 생성

[F] 정리 결과 리포트
  - "중복 45장 제거, 354장 시간순 정렬, 인물 5명 분류 완료"
  - 절약된 용량 표시

  - GET /api/photos/report — 정리 리포트
